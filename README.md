
# Blackjack

A simple blackjack card game implementation.

## Getting Started

1. Clone the repository
2. Install dependencies
3. Run the game

## Rules

- Aim to get closer to 21 than the dealer
- Cards 2-10 are worth face value
- Face cards (J, Q, K) are worth 10
- Aces are worth 1 or 11

## Usage

```bash
python blackjack.py
```
